Project Name eCommerce Site
Description Of The Project
I developed this site at the time of web development class.
This site is an eCommerece site of a bakery.
I developed this site with the help of HTML, CSS, Bootstrap, JavaScript.
This is a fully functional demo site. We can chose products, can add to cart, can make payment, can place order also we can apply for taking the franchise of this company.
Technical Description
To develope the body of this site I used the scripting language HTML.
For the designing of the UI I used the Scripting Language CSS & Bootstrap.
All the Icons which is used, I brought these from FontAwesome.
To make this site dynamic I used the programming language JavaScript.
Here I have no backend, So for the back end I used the super power of browser i.e. Session Storage.
Here I stored all the data which is chosed by the user and retrived the data from the Session Storage at the time of making payment.
For make payment I used Stripe. Once the user make payment then the data automatically removed from the Session Storage.
Finally I deployed this site in Firebase That's my site works.
